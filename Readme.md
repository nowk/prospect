## Installation

    .gitignore_global -> ~/prospect/git/gitignore_global
    .ondirrc          -> ~/prospect/ondir/ondirrc
    .tmux.conf        -> ~/prospect/tmux.conf
    .vim              -> ~/prospect/vim
    .vimrc            -> ~/prospect/vim/vimrc
    .zprofile         -> ~/prospect/zprofile
